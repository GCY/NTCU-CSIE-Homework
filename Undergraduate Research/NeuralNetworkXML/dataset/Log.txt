用於訓練類神經網路的資料集(Dataset)
放置於Arm_action_training\data\ 資料夾下
檔名為"arm_action.dat"

資料格式
	1.每一行代表每一筆資料
	2.每一筆資料中每個數據之間都用空格分隔
	3.前面放輸入資料，後面放預期輸出
	
	4.以Pattern Recognition(分類器)的為例
		若有3項輸入值，要分為5種目標類別
		輸入值照順序填上，以空白間隔
		代表5種類別的數據分別為:
			(1) 1 0 0 0 0
			(2) 0 1 0 0 0
			(3) 0 0 1 0 0
			(4) 0 0 0 1 0
			(5) 0 0 0 0 1
			
		而每筆數據格式如下:
			50.321 75.100 90.524 1 0 0 0 0
			25.652 39.682 11.023 0 1 0 0 0
			68.241 02.356 88.213 0 0 1 0 0
		   (輸入1)(輸入2)(輸入3)(目標類別)

1. arm_action V1.dat
	用於訓練 neural_network V1.xml
	4個輸入(左肘 左肩 右肘 右肩)
	5個目標類別
		(1) 雙手向上伸直
		(2) 雙手水平伸直
		(3) 雙手水平伸直後向上彎曲90度
		(4) 雙手水平伸直後向下垂45度
		(5) 雙手向下伸直
